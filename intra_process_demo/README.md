## **Build**

```bash

mkdir -p demo_ws/src
cd ~/demo_ws/src
git clone https://github.com/ros2/demos.git
cd ~/demo_ws
source /opt/ros/<DISTRO_NAME>/setup.bash
# source /opt/ros/rolling/setup.bash
colcon build --package-select intra_process_demo
```

```bash

cd ~/demo_ws
source install/setup.bash
```
